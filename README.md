# Programming Challenges Hub | مركز تحديات البرمجة

Welcome to the **Programming Challenges Hub**, an open-source bilingual project  
(English + Arabic) built for students, developers, and problem-solving lovers.

مرحبًا بك في **مركز تحديات البرمجة**، مشروع مفتوح المصدر  
ثنائي اللغة (عربي + إنجليزي) موجّه للطلاب والمبرمجين ومحبي حلّ المشكلات.

---

## Project Purpose | هدف المشروع

- Provide a structured collection of programming problems  
- Offer solutions in multiple programming languages  
- Help beginners practice algorithms & data structures  
- Build a community-driven open-source hub  
- Encourage pull requests and collaboration  

- توفير مجموعة من مسائل البرمجة بشكل منظّم  
- تقديم حلول بعدة لغات برمجة  
- مساعدة المبتدئين على ممارسة الخوارزميات وهياكل البيانات  
- بناء مشروع مجتمعي مفتوح المصدر  
- تشجيع المساهمات وطلبات السحب (Pull Requests)

## How to Contribute | كيف تساهم؟

You can contribute by:

- Adding new problems  
- Adding solutions in any language  
- Improving existing solutions  
- Translating problems  
- Adding tests  

يمكنك المساهمة عبر:

- إضافة مسائل جديدة  
- إضافة حلول بأي لغة  
- تحسين الحلول الموجودة  
- ترجمة المسائل  
- إضافة اختبارات

Make sure to read the **CONTRIBUTING.md** file.
