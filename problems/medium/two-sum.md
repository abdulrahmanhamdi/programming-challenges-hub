# Two Sum | مجموع رقمين

## Description
Given an array of integers and a target value, return indices of two numbers whose sum equals the target.

## الوصف
أعطِ مصفوفة أعداد صحيحة ورقم هدف، وأعد فهارس رقمين مجموعهما يساوي الهدف.

## Example
Input: nums = [2, 7, 11, 15], target = 9  
Output: [0, 1]

## Difficulty
medium
