# Move Zeros | نقل الأصفار إلى النهاية

## Description
Move all zeros in an array to the end while keeping the relative order of non-zero elements.

## الوصف
انقل جميع الأصفار داخل المصفوفة إلى النهاية مع الحفاظ على ترتيب العناصر غير الصفرية.

## Example
Input: [0,1,0,3,12]  
Output: [1,3,12,0,0]

## Difficulty
medium
