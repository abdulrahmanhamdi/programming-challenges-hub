# Contributing Guide | دليل المساهمة
Thank you for contributing!
شكرًا لرغبتك في المساهمة!
---
## 1. Adding a New Problem | إضافة مسألة جديدة
Choose the correct folder:

```
problems/easy/
problems/medium/
problems/hard/
```

Create a file like:

```
reverse-string.md
```

Use this format | استخدم هذا النموذج:

```
# Problem Title | عنوان المسألة

## Description | الوصف
Write the problem description here.
اكتب وصف المسألة هنا.

## Examples | أمثلة
Input:
Output:

## Constraints | القيود
- ...

## Difficulty | الصعوبة
easy | medium | hard
```

---

## 2. Adding a Solution | إضافة حل

Go to:

```
solutions/python/
solutions/javascript/
solutions/java/
solutions/cpp/
```

Add a file named after the problem:

```
reverse-string.py
```

---

## 3. Update contributors.json | تحديث ملف المساهمين

```
{
  "name": "Your Name",
  "github": "your-username"
}
```

---

## 4. Pull Request Rules | قواعد الـ Pull Request
- One change per PR
- Clear commit messages
- Code must be clean and runnable
- تعديل واحد لكل Pull Request
- رسائل التزام (commit messages) واضحة
- الكود يجب أن يكون نظيفًا ويعمل بدون أخطاء

Happy coding!
برمجة ممتعة!
